# User Stories

## Authentication
*   **US-01:** Como usuario, quiero registrarme con mi email y contraseña para acceder a la plataforma.
*   **US-02:** Como usuario, quiero iniciar sesión para ver mis datos y pedidos.

## Store Management
*   **US-03:** Como vendedor, quiero crear mi tienda para empezar a vender.
*   **US-04:** Como vendedor, quiero agregar productos (foto, precio, stock) para armar mi catálogo.

## Shopping Experience
*   **US-05:** Como comprador, quiero buscar productos por nombre para encontrar lo que necesito.
*   **US-06:** Como comprador, quiero agregar productos de diferentes tiendas al carrito para hacer una sola compra.
*   **US-07:** Como comprador, quiero finalizar mi compra y recibir un número de pedido.

## Administration
*   **US-08:** Como admin, quiero aprobar tiendas nuevas para asegurar la calidad del marketplace.
